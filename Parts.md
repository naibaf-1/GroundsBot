## Calculation
#### Chins
Platz: 30cm
Puffer: +/-1cm
=> Platzbedarf: 28cm
	=> Höhe: 7cm
		=> Umfang (Kreis): 7cm * pi = 22cm
		=> Abstand: 28cm - 7cm = 21cm
=> Gesamtlänge: 2 * 21cm + 22cm = 64cm
	=> Beide Ketten zusammen: 64cm * 2 = 128cm
		=> Problem: Eine Kette ist 78 cm lang (Ersatz/Puffer: 78cm - 64cm = 14cm)
